# 获取 浏览器标签页 网站icon

**网页图标 - favicon.ico**

在网站主页链接后加 ==/favicon.ico== 即可，不是所有网站都有

eg:
```
https://readpaper.com/favicon.ico
https://github.com/favicon.ico
https://www.tongji.edu.cn/favicon.ico
https://grad.illinois.edu/favicon.ico
```
