# 配置 SHELL

## 查看 SHELL

查看当前 SHELL

```bash
lzy@legion:~ $ echo $SHELL
/bin/bash
```

系统支持的 SHELL

```bash
lzy@legion:~ $ cat /etc/shells
# /etc/shells: valid login shells
/bin/sh
/bin/bash
/usr/bin/bash
/bin/rbash
/usr/bin/rbash
/usr/bin/sh
/bin/dash
/usr/bin/dash
```
