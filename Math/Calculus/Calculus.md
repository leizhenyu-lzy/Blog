# Integration

## Integration by Substitution - 积分换元法

$$\int_{\alpha}^{\beta} f[\varphi(x)] \varphi'(x)dx \quad \xrightarrow{\text{令 } t=\varphi(x)} \quad \int_{\phi(\alpha)}^{\varphi(\beta)} f(t)dt = \int_{a}^{b} f(t)dt$$

$$a = \varphi(\alpha), b = \varphi(\beta)$$
