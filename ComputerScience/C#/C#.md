# 安装 .NET Core SDK

Ubuntu 下使用 snap 应用商店 进行下载

```bash
lzy@legion:~/Project/Blog$ snap install dotnet-sdk --classic
dotnet-sdk 7.0.304 from Microsoft .NET Core (dotnetcore✓) installed
```

查看版本

```bash
lzy@legion:~$ dotnet --version
7.0.304
```

创建工程 "**dotnet new console**"

运行 "**dotnet run**"
