## 01 - Linux系统介绍

Linux
1. 多用户 - 支持多个用户同时登录和操作(远程登录)
2. 多任务 - 每个用户可以运行多个程序
3. 一切皆文件

[linux-目录配置](../../../Linux/LinuxSystemic.md#linux-目录配置)

指令 - `命令 [选项] [操作对象]`

`pwd`   - print current working directory

`ls`    - list directory contents
1. `ls [Path]` eg : `./`, `../`, `/home`
2. `ls [Code]` eg : `-l` 列表形式, `-a` 显示所有(包括隐藏文件), `-h` 较高可读性
