# 堡垒机

## 堡垒机是什么？

堡垒机(堡垒服务器，运维审计系统)，用于管理&控制敏感系统和服务器
1. 保护 敏感数据资源 & 网络资源
2. 免受 黑客、内部威胁、未经授权的访问

堡垒机 由 跳板机(前置机) 的概念演变而来

**主要功能** (4A)
1. 认证 authentication
2. 账号 account
3. 授权 authorization 
4. 审计 audit 事间监察

事前授权、事间监察、事后审计

**工作原理**
1. 设立安全的中转服务器，实现对敏感系统的访问控制
2. 用户需要首先通过堡垒机进行身份认证，包括用户名、密码、双因素认证等方式
3. 通过认证后，用户可通过堡垒机访问授权目标服务器，堡垒机充当用户和目标服务器之间的中转角色
4. 堡垒机会监控用户的操作行为，防范恶意行为，并记录供审计使用，便于安全团队进行调查和追溯

**应用领域**
1. 企业内部安全 : 用于管理和控制内部员工对敏感系统和服务器的访问，防止内部人员账号滥用、权限滥用、数据泄漏
2. 云安全 : 用于管理云服务器的访问，提供额外的安全保护，确保云上的敏感数据





# 红区 黄区 绿区

## CORE区 DMZ区 PTR区 红区 绿区 蓝区

[CORE区 DMZ区 PTR区 红区 绿区 蓝区](https://www.jianshu.com/p/3db1852623f9)

解释
1. CORE指核心区
   1. 位于企业内网
   2. 部署在CORE区的应用不允许从互联网直接访问，也无法直接访问互联网
2. DMZ指资源隔离区
   1. CORE区的应用需要访问互联网ip（比如调用公网接口），需要在DMZ区搭出向代理，通过DMZ区访问互联网ip
   2. 互联网ip想要访问CORE区应用，需要在DMZ区搭入向代理，通过DMZ区访问CORE区应用
3. PTR指合作伙伴区
   1. 走的是专线，CORE区的应用需要访问其他租户或者云以外的内网ip（比如调用其他租户的接口），需要在PTR区搭出向代理，通过PTR区访问内网ip
   2. 其他租户或者云以外的内网ip想要访问CORE区应用，需要在PTR区搭入向代理，通过PTR区访问CORE区应用。
   3. （新规则：不同租户之间，可以直接CORE区访问CORE区，不需要PTR）
4. 专线
   1. 企业之间点对点拉一条独立的网线，只有开通专线的两个企业可以互相访问，安全性高，但价格昂贵
   2. 专线可以是物理信道也可以是虚拟信道
      1. 如果是物理信道，就是拉一条网线
      2. 如果是虚拟信道，就是在一般的信道上为用户保留一定的独享带宽
5. 红区相当于CORE区
6. 绿区相当于DMZ和PTR区
7. 蓝区用于部署测试环境应用





## 通讯行业数据防泄露解决方案

[通讯行业数据防泄露解决方案](https://zhuanlan.zhihu.com/p/497271844)

网络架构分为红区、黄区、绿区，每个区域之间都是独立的网络环境，各个区域之间文件交互有特定网络通道交互数据
1. 红区
   1. 主力产品、关键技术、核心代码等关键信息
   2. 项目部门或产品部门有独立的网络环境，专属的服务器，通过虚拟桌面办公
   3. 文件到黄区与绿区已实现严格的文档输出管控，无需再重复进行文档安全管控
   4. **适用部门**：各产品线研发部门及有密切业务关系的业务部门或业务角色，如质量部、工程部、技术服务中心等
2. 黄区
   1. 一般产品技术平台的规划、设计、开发、文档、测试环境等
   2. 有独立的网络环境，文件到红区只能通过单项FTP(File Transfer Protocol)
   3. 到绿区走EIP(Enterprise Information Portal 企业信息门户)权限控制
   4. 黄区可通过远程桌面文件隔离方式访问绿区，访问业务系统有ERP、EIP、内网邮箱、UCA、云桌面等
   5. **适用部门**：计划、采购、工程、基板、合同、财务等部门
3. 绿区
   1. 少数研发测试、文档办公、对外交流、技术支持、移动办公环境
   2. 有独立的内网环境并可以访问外网
   3. 文件到红区只能通过单项FTP，通过单向外网邮箱到黄区，可通过远程桌面文件隔离方式访问黄区
   4. 访问业务系统有EIP、外网邮箱、CRM、UCA等，并会使用QQ、微信等跟外部沟通，目前到绿区的文件无管控措施
   5. 适用部门：市场、技术支持、销售服务等部门
4. 外部访问
   1. 外部通过互联网VPN连到公司绿区网络，会根据VPN登陆的用户限定访问业务系统EIP的权限
   2. 黄区用户VPN登陆之后不能直接登陆EIP，需要访问黄区业务系统需要远程桌面到黄区电脑操作
   3. 绿区电脑用户VPN登陆之后等同于公司绿区电脑
