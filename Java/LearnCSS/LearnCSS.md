## 快速入门

style

分号结尾

语法：
```html
选择器
{
    声明1;
    声明2;
    声明3;
}
```

HTML与CSS结合方法：
1. 可以直接在html中内嵌
   ```html
   <head>
    <meta charset="UTF-8">
    <title>test1</title>

    <!--可以在这里编写CSS代码-->
    <style>
        h1
        {
            color: crimson;
        }
    </style>
   </head>
   ```
2. 另写一个CSS文件，在html中进行关联
   ```html
   <link rel="stylesheet" href="test1.css">
   ```
