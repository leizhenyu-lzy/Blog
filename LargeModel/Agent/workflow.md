# Coze

[Coze 旧版 - Website](https://www.coze.cn/home)

[Coze 新版 - Website](https://code.coze.cn/home)


模式
1. 对话流
   1. 单 Agent (自主规划模式)
      1. 自主决定调用的 附加功能 (工作流 & 插件 & 知识库 & 记忆)
      2. 可以 添加调用逻辑 到 system prompt 中
   2. 单 Agent (对话流模式)
      1. 编排节点 (意图识别 & 选择器 & 知识库 & 总结 & API)
      2. 相当于 在对话中 调用 工作流
   3. 多 Agents
2. 工作流 : 没有对话上下文
   1. 可以 直接运行
   2. 也可以 作为 能力 添加到 agent 中
   3. 一般是 通过 API / ChatSDK 方式 发布


发布渠道
1. 豆包
2. 飞书(多维表格)
3. 微信(小程序 / 服务号 / 订阅号)
4. API
5. Chat SDK


任务
1. 批量任务
2. 异步任务
